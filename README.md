# Autonomous Supply Chain Resilience & Optimization System (ASCROS)

## 📌 Project Overview
This project focuses on building an AI-driven system to optimize inventory management and reduce stockouts using demand analysis and simulation.

## 🚀 Features
- Demand and sales trend analysis
- Inventory tracking and stockout detection
- AI-based ordering decision simulation
- Interactive Power BI dashboard

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- Power BI
- Simulation Modeling

## 📊 Dashboard Insights
- KPI Metrics (Profit, Demand, Sales, Lost Sales)
- Demand vs Sales trends
- Inventory performance over time
- Stockout (lost sales) analysis
- AI decision behavior visualization

## 🎯 Objective
To optimize supply chain decisions and improve business efficiency through data-driven insights.
